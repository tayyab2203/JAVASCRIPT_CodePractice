Arrays

Arrays are a collection of items stored at contiguous memory locations. They are used to store multiple values in a single variable, which can be of the same type.

Creating an Array

In many programming languages, you can create an array by specifying the type and the number of elements it will hold. For example, in Python, you can create an array using the `array` module or simply use a list, which is a dynamic array.

import array as arr my_array = arr.array('i', [1, 2, 3, 4, 5]) # 'i' indicates an array of integers print(my_array)

Accessing Array Elements

Array elements can be accessed using their index, which starts from 0. For example:

print(my_array[0])  # Output: 1 
print(my_array[2])  # Output: 3
